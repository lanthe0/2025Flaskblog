# 2025Flaskblog
2025小学期作业

## 快速开始（本地部署）

### 1. 克隆项目
```bash
 git clone https://github.com/lanthe0/2025Flaskblog.git
 cd 2025flaskblog
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 运行项目
```bash
# 开发环境
python app.py
# 或生产环境（支持IPv6）
python run_ipv6.py
```

### 4. 访问
浏览器打开:  http://127.0.0.1:5000  （或终端输出的地址）

---

如遇依赖或数据库问题，请先检查 Python 版本（经过测试，3.8.20与3.11均可），并确保 requirements.txt 安装无误。
